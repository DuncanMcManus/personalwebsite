projects

Rhode to Vote

Rhode to Vote is a ride-sharing platform for voting access. People who can't get to the polls are able to sign up and request a ride. Volunteers sign up and are given a route to pick up voters based on available time, distance, and preferences. On this 4 person multi-month project, I worked on the Java backend. I wrote three different modified Dijkstra and Breadth First Search graph traversal algorithms to find optimal routes.

AnyWays

Google Maps clone made from scratch. Web app displays a street map that can be panned, zoomed, and clicked for point-route selection. I wrote the Java backend which implements a KD-Tree to store location data, A* for finding shortest paths, Proxy Pattern for caching on database requests, and a Trie for autocorrecting street names. I also wrote a JavaScript front-end which uses tile caching for displaying map sections.

Attic

Attic gives users ownership of a Virtual Reality space which they can create, design, and share with others. For Hack@Brown 2018 I led my team in creating a mobile AR web app that lets the user select a 3D model from a menu and place it in their surrounding space. This is an ongoing project with a POC written in JavaScript using Google's three.ar.js, it is being transitioned to Unity.

Soundstage

Created for Hack@Brown 2017. Soundstage is for music fans that have trouble keeping up with all their favorite artists. After an artist name is entered, it presents a chronological feed of releases. I was the team leader and wrote JavaScript to get user input, make a request to the Spotify API, and parse results.
